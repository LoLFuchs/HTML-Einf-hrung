Aufgabe 1

Was ist HTML und wofür steht die Abkürzung?
  • HTML steht für HyperText Markup Language
  • Grundlegende Struktur von Webseiten
  • Es definiert die Struktur eines Dokuments (Überschriften, Absätze, Links, Bilder usw.)
  • HTML gibt dem Browser Anweisungen, wie Inhalte angezeigt werden sollen
Die Geschichte von HTML: Wer hat es erfunden?
  • HTML wurde von Tim Berners-Lee entwickelt
  • Erfunden im Jahr 1989, öffentlich eingeführt 1991
  • Entwickelt im CERN, um wissenschaftliche Dokumente zu verknüpfen
  • Seine Vision: Ein vernetztes Informationssystem für alle
Die Evolution von HTML: Von HTML 1.0 zu HTML5
  • HTML 1.0 (1993): Grundstruktur für Webseiten
  • HTML 2.0 (1995): Verbesserte Formulare und Eingaben
  • HTML 3.2 (1997): Neue Elemente wie Tabellen und Applets
  • HTML 4.01 (1999): Unterstützung für Stylesheets und verbessertes Design
  • HTML5 (2014): Multimediale Funktionen, native Unterstützung für Audio und Video, Flexibilität für mobile Geräte
Das World Wide Web Consortium (W3C)
  • W3C: Standardisierungsorganisation für das Web, gegründet von Tim Berners-Lee
  • Ziel: Webtechnologien wie HTML, CSS, XML zu entwickeln und zu pflegen
  • Wichtig: Sicherstellung der Interoperabilität von Webtechnologien und Förderung von Innovation
Hyperlinks und die Revolution des Internets
  • Hyperlinks: Verbindungen zwischen verschiedenen Dokumenten oder Webseiten über das <a>-Tag
  • Revolutionär, da sie nahtlose Navigation und Verlinkung von Inhalten ermöglichen
  • Fundament für das Konzept des World Wide Web: Ein global vernetztes Informationssystem
